# Service Workflow

## Test Drive Reservation

1. User opens the catalog.
2. User selects a car option.
3. User selects a service center.
4. User selects a reservation date.
5. User logs in with Kakao.
6. The app stores the Kakao user if needed.
7. The app creates a test-drive reservation.

## Admin

1. Admin logs in.
2. Admin views reservations or product options.
3. Admin updates or deletes reservations.
4. Admin creates brands, models, and car options.
